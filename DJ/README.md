# DJ Mixing Deck Workshop

This workshop is organized into 3 progressive parts, each building on the previous one.

## Structure

```

│   ├── sketch.js           # Reference code

│   ├── assets/             # Sound files
│   └── workshop/           # Workshop documentation
│       ├── workshop.md     # Main workshop guide
│       ├── detailed.md     # Detailed step-by-step guide
│       ├── guide_tldr.md   # Quick reference
│       └── workshop_metadata.md
│
├── part2_customization/    # Part 2: Adding Customization
│   ├── sketch.js           # Reference code
│   ├── index.html          # HTML file
│   ├── assets/             # Sound files
│   └── workshop/           # Workshop documentation
│       ├── workshop.md     # Main workshop guide
│       ├── detailed.md     # Detailed step-by-step guide
│       ├── guide_tldr.md   # Quick reference
│       └── workshop_metadata.md
│
└── part3_advanced/         # Part 3: Advanced Features
    ├── sketch.js           # Reference code
    ├── index.html          # HTML file
    ├── assets/             # Sound files
```

## Parts Overview

### Part 1: Starter
**Basic DJ Mixing Deck**
- Two play/pause buttons
- Two volume sliders
- Basic mixing functionality
- Simple, clean interface

### Part 2: Customization
**Adding Customization Features**
- File upload for background images
- File upload for track sounds
- Mobile-friendly design
- Touch support
- Responsive layout

### Part 3: Advanced
**Advanced DJ Features**
- Time sliders (seek/jump in tracks)
- Crossfader for smooth transitions
- BPM visualization (pulsating circles)
- All customization features from Part 2
- Refactored code with helper functions

## How to Use

1. **Start with Part 1**: Follow the workshop in `part1_starter/workshop/`
2. **Progress to Part 2**: Build on Part 1 by adding customization
3. **Complete with Part 3**: Add advanced features and learn code organization

Each part includes:
- **Reference code** (`sketch.js`) - Complete working example
- **Workshop documentation** (`workshop/`) - Step-by-step guides
- **Assets** (`assets/`) - Sound files needed for the project
